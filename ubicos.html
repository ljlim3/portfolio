What's UbiCoS?


Ubiquitous Collaboration Support (UbiCoS) is an intelligent educational platform that supports and facilitates students' help-giving interactions through different learning tools.

Teacher Dashboard


Through the teacher dashboard, teachers would like to understand student participation. This is achieved by showing 1) for each activity, which students participated/not participated 2) for each student, in which activities they participated/not participated 3) what badges did each student earn 4) for what contributions did the students earn the badges, without having to dig deep in order to see these information.
